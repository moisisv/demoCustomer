## üèÅ Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/moisisv/demoCustomer.git
   ```
2. Navigate to the project directory:
   ```bash
   cd DemoPage
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the local development server:
```bash
npm run dev
```
